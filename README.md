# SMS spam detection using machine learning


To Run this app

- ```git clone https://github.com/SharathHebbar/SMS-spam-detection-using-machine-learning.git```
- ```pip install -r requirements.txt```
- ```python server.py```

Then navigate to

- ```http://127.0.0.1:5000```
